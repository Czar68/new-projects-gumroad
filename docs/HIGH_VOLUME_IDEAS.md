# High-volume spreadsheet product ideas (Phase 2A)

**Criteria:** Proven demand, $10–$50, 1,000+ sales potential, minimal dependencies, beginner→intermediate value.

---

## 1. E-commerce Profit & Pricing Calculator (Shopify / Etsy / eBay)

| Factor | Detail |
|--------|--------|
| **Demand** | Sellers need COGS, fees, shipping → net profit. Gumroad/Etsy products $29–49; high volume. |
| **Structure** | Inputs: cost, price, platform fee %, shipping. Outputs: profit, margin %, break-even, ROI. Multi-platform (Shopify, Etsy, eBay) in one sheet. |
| **Price** | $27–$47. |
| **Potential** | 1,000+ (e-commerce seller base is huge). |
| **Dependencies** | None (Google Sheets + formulas). |
| **Value** | Beginner: "What will I make?" Intermediate: Compare platforms, bulk paste rows. |
